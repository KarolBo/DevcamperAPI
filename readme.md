# DevCamper API

> Backend API fo DevCamper application, whick is a bootcamp directory website.

## Usage

Rename "config_template.env" to "config.env" and update the values to your own

## Install dependencies

```
npm install
```

## Run App
```
# Run in dev mode
npm run dev

## Run in prod mode
npm start
```
